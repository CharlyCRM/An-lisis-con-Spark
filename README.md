# Analisis de Datos con Spark
Se realiza un análisis de datos sobre los resultados académicos de un centro de estudio, con el objetivo de observar diferencias entre el nivel educativo de los padres y el resultado de los alumnos en alguna de las materias, además de observar si la raza etnica, sexo o almuerzo afectan a los resultados.
